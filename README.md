# Encryption and Decryption Simulator

This project is a visual guide to secure communication, demonstrating Symmetric and Asymmetric encryption techniques.

## Project Structure

- **`/frontend`**: Contains the React + Vite application for the visual simulator.
- **`/backend`**: Placeholder for future server-side implementation and API logic.

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Running the Simulator
1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies (if not already installed):
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
